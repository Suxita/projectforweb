## рЃЋрЃћрЃЉ рЃЊрЃљрЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў

---

**рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ рЃЌрЃАрЃБрЃА рЃарЃћрЃфрЃћрЃърЃбрЃћрЃЉрЃўрЃА рЃАрЃљрЃўрЃбрЃќрЃћ**

рЃЕрЃЋрЃћрЃю рЃерЃћрЃЋрЃљрЃњрЃарЃЮрЃЋрЃћрЃЌ рЃАрЃўрЃљ рЃўрЃЏ рЃАрЃљрЃГрЃЏрЃџрЃћрЃЉрЃўрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃА рЃбрЃарЃБрЃџ рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЊрЃдрЃћрЃћрЃЉрЃерЃў рЃљрЃџрЃЉрЃљрЃЌ рЃЊрЃљ рЃЊрЃљрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ ­Ъўї

рЃЏрЃўрЃюрЃЊрЃљ рЃњрЃљрЃЕрЃЋрЃћрЃюрЃЮрЃЌ рЃАрЃљрЃўрЃбрЃўрЃА рЃърЃарЃћрЃЋрЃўрЃБ:
![рЃАрЃљрЃўрЃбрЃўрЃА рЃърЃарЃћрЃЋрЃўрЃБ](showcase.gif)

рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃў рЃўрЃЏ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃћрЃЋрЃюрЃћрЃЉрЃўрЃАрЃљ рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃўрЃЦрЃюрЃљ рЃЊрЃљрЃЎрЃЏрЃљрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃБрЃџрЃў:

1. рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў рЃАрЃљрЃўрЃбрЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў W3C рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ:
2. Javascript-рЃўрЃЌ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃърЃћрЃњрЃўрЃюрЃљрЃфрЃўрЃљ

3. рЃцрЃЮрЃарЃЏрЃљрЃАрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃерЃў
4. рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃерЃў (рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃЉрЃљрЃќрЃљрЃАрЃЌрЃљрЃю рЃЋрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃЊрЃў рЃЏрЃљрЃњрЃарЃљрЃЏ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЊрЃљрЃЋрЃгрЃДрЃЋрЃўрЃбрЃћ рЃерЃћрЃЏрЃћрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃўрЃюрЃљ)
5. transform рЃЊрЃљ transition-рЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ
